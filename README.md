# online-courses-app
Interactive web application displaying online courses

# Overview

The Online Courses Explorer is an interactive web application developed for the Department of Computer Science. The app allows users to explore a list of online courses, sort and filter them based on various criteria, and register for courses using a built-in sign-up form with validation.

This project was developed as part of the M.Sc AIML Online – Full Stack Development course.

# Features

Dynamic Course Display
Fetches course data from a JSON file (simulating an API).
Displays course information including title, instructor, category, duration, rating, price, and enrollment count.
Renders course cards dynamically on the page.
Filter & Sort Functionality

Users can filter courses by category.
Courses can be sorted by rating or price.
Responsive and easy-to-use dropdown controls.
Sign-Up Form with Validation

Users can register for a course by filling out the sign-up form.
Form validation ensures all required fields are filled correctly before submission.

Enhanced Presentation
Clean, modern UI with centered headers and well-structured course cards.
Responsive layout compatible with multiple screen sizes.
